Sketch #6: Coffee Run

If coffee machine working at home then drink coffee at home otherwise go out for a coffee run

What rule did you design? The rule is that the character can only move toward the coffee shop if the "machine status" is broken, which is triggered by holding down the mouse.

What is the “otherwise”? Otherwise, when the mouse is released, the machine is considered 'working' and the character is instantly snapped back to the Home zone (x = 50).

How does the user know the state changed? The state change is communicated through on-screen status text that changes color, and the physical constraint of the circle being 'locked' or 'unlocked' for movement.
