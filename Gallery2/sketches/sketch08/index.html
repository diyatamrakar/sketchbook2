Sketch #8: Multi-State Fruit Gallery

What determines movement? Movement is triggered by the keyPressed() function, specifically listening for the Left and Right arrow keys to change the currentIndex.

How are boundaries handled? I used wrap logic to allow infinite navigation; if the user goes past the fifth fruit, the system resets the index to 0.

Reflection: This sketch models a chromatic progression. By navigating through the fruits, the user experiences a shift in the color spectrum from cool blues to warm oranges and deep reds. I learned how to manage an array of external media and ensure they scale responsively to the window size.
