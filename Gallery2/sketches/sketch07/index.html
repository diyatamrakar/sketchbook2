Sketch #7: Uneven Rules

What inequity does this system model? This models labor inequality. Both circles receive the same input (mouse click), but one has "fixed efficiency" while the other suffers from "energy decay" or fatigue, making progress harder for the second agent over time.

How does time make the difference more visible? At the start, the circles move together. However, as the user continues to interact, the Red circle slows down significantly due to the accumulation of fatigue, creating a visible gap between the two outcomes.

Reflection: I learned that simple math (subtracting a tiny amount from a variable) can effectively change the "feel" of a system from fair to unfair without needing complex visuals.
